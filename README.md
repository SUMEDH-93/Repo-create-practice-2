# 🚀 Git & GitHub Practice Project

## 📌 About Project

This project was created for learning and practicing Git and GitHub commands.

In this project, we learned:

* How to initialize a Git repository
* How to create and manage branches
* How to commit changes
* How to push code to GitHub
* How to merge branches
* How to use stash and rebase commands

---

## 🛠️ Technologies Used

* Git
* GitHub
* VS Code

---

## 📂 Features Practiced

✅ Git Init
✅ Git Add & Commit
✅ Git Push & Pull
✅ Branch Creation
✅ Merge Branches
✅ Git Stash
✅ Git Rebase

---

## ▶️ Commands Used

```bash
git init
git add .
git commit -m "first commit"
git branch feature/video
git checkout feature/video
git push origin main
```

## 🎯 Purpose

The main purpose of this project is to improve Git and GitHub skills through hands-on practice.
